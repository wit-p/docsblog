AppSheet は以前からスプレッドシートのデータからアプリを構築できました。
それとは何がどう違うのか、
Spec を使う場面はどういった時なのでしょう。<br/>

Spec 機能とはなにかを調べ、実際に日記アプリ作成を通して Spec をどういった場面で使えるのかを考えてみます。<br/><br/>

## 1. Spec 機能とはなにか

Spec は [AppSheet とはなにか](https://witpg.gatsbyjs.io/appsheet/about)に記述した[3 つの特徴のうちのインテントアウェアに相当する機能](https://witpg.gatsbyjs.io/appsheet/about#2-2-appsheet-の主な-3-つの特徴)に相当します。
スプレッドシートのデータからアプリを構築する方法は

- [習慣化アプリの使い方](https://witpg.gatsbyjs.io/appsheet/howto)
- [SQ4R アプリの作り方](https://witpg.gatsbyjs.io/appsheet/sq4r)

で実践しています。<br/>

これらの方法ではスプレッドシートを用意し、そのデータから AppSheet アプリケーションエディタで詳細を設定しアプリを構築していくものでした。<br/>

私たちは Alexa や Siri で私生活や家の管理を簡単にできるようになってきました。それらと同じように仕事に必要なアプリを検索フォームに簡単な文章や言葉を入力して作成できたら便利ですよね。<br/>

Spec では最初にスプレッドシートを用意することなく Google 検索フォームに知りたいことを入力するようにアイデアからアプリを構築できます。<br/><br/>

## 2. 日記アプリのアイデア

### おおまかに作りたいアプリのアイデアを書き出す

- 日記を書き始めようとおもっている
- 日記帳はプライベートの出来事・仕事の出来事・ペットの出来事で使い分けたい
- 日記には、日記のタイトル、日付、日記内容、その日に印象に残った写真を一枚を保存したい

## 3. 今回作るアプリのフレーズ一覧

### アイデアからアプリに必要な機能を書き出す

この表は、[4. Spec 機能を使い日記アプリを作成](https://witpg.gatsbyjs.io/appsheet/spec#4-spec-機能を使い日記アプリを作成)に使用します。
スライドショーを見ながら日記アプリを作成する際に、コピー&ペーストするなどしてお役立てください。<br/>

日記アプリにアイデアを追加したり、
オリジナルアプリに挑戦するときにはその場のアイデアの思い付きで色々試すのも面白いですし、
このように予めおおまかな設計をしておくのも良いかもしれません。

| 日記帳の作成             | 対応する Spec のフレーズ             |
| ------------------------ | ------------------------------------ |
| 日記の一覧タブを作成     | this app has a collection of Diaries |
| 日記の項目に年月日を登録 | a Diary has a Date property          |
| 日記に登録するの他の項目 | ※ 自動生成した項目をそのまま使う     |
| 日記を新規作成           | user can add new Diaries             |
| 日記を編集               | user can edit Diaries                |
| 日記を削除               | user can delete Diaries              |

| 日記帳の種類の作成         | 対応する Spec のフレーズ                |
| -------------------------- | --------------------------------------- |
| 日記帳の種類一覧タブを作成 | this app has a collection of Categories |
| 日記帳の種類に登録する項目 | ※ 自動生成した項目をそのまま使う        |
| 日記帳の種類を新規作成     | user can add new Categories             |
| 日記帳の種類を編集         | user can edit Categories                |
| 日記帳の種類を削除         | user can delete Categories              |

| その他の機能                                                      | 対応する Spec のフレーズ                      |
| ----------------------------------------------------------------- | --------------------------------------------- |
| 日記と日記帳の種類を紐付け<br/>日記を書くときに日記帳の種類を選択 | a Category has many Diaries                   |
| 日記をカテゴリー別に一覧表示するタブを作成                        | user can view all Diaries grouped by Category |
