## 1. Google スプレッドシートの編集

### 1-1. まずはデータ設計をする

| NAME       | TYPE     | REQUIRE  | INITIAL VALUE |
| :--------- | :------- | :------- | :------------ |
| 日付       | DateTime | 入力必須 | NOW()         |
| タイトル   | Text     | 入力必須 |               |
| 調査       | LongText |          |               |
| 質問       | LongText |          |               |
| 読み       | LongText |          |               |
| 回答       | LongText |          |               |
| 記録       | LongText |          |               |
| 復習       | LongText |          |               |
| ステータス | Enum     | 入力必須 |               |

### 1-2. 次にスプレッドシートを編集する

[Google ドライブにログイン](https://drive.google.com/)してスライドを参考に実際にやってみましょう。

https://slides.com/wit-m/sq4r-spreadsheet

スプレッドシートの編集ができました。<br/>
ここからは、習慣化アプリの AppSheet に編集・追加していきます。

## 2. Google AppSheet にログイン

[ここから AppSheet にアクセス](https://www.appsheet.com/)してログインしましょう。

https://slides.com/wit-m/appsheet-login

## 3. AppSheet の編集・追加

### 3-1. Data タブの設定をする

最初に、スプレッドシートに新規追加した sq4r シートを AppSheet に読み込ませます。
データ設計を元に設定していきましょう。<br/>
終わったら SAVE しておきましょう。

https://slides.com/wit-m/sq4r-data

### 3-2. UX タブの設定をする

sq4r アプリの使い勝手を向上させていきましょう。

https://slides.com/wit-m/sq4r-ux

最後に SAVE して SQ4R アプリが完成です！おつかれさまでした。
