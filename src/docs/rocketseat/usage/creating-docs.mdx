docs フォルダー内に作成されたすべての MDX ファイルは、ファイル名で定義されたパスを持つページになります。すべてのドキュメントファイル内で、次のオプションを定義できます。

| Key                    | Required | Description                |
| ---------------------- | -------- | -------------------------- |
| title                  | Yes      | No                         |
| description            | No       | Page description           |
| image                  | No       | Used for og:image          |
| disableTableOfContents | No       | Disables Table Of Contents |

## 例

```mdx
---
title: "My Example"
description: "A simple description for this page"
image: /banner.jpeg
disableTableOfContents: false
---

Wow, this is a nice page
```

## ファイルの埋め込み

ドキュメントの Web サイトでは、よく知られているサービス（CodeSandbox、Twitter など）を埋め込む必要がある場合があります。
あなたがそれをする必要があるならば、あなたは単にリンクをコピーして貼り付けることができます（素晴らしい[gatsby-remark-embeder](https://www.gatsbyjs.com/packages/gatsby-remark-embedder/)によって供給されます）。
[サポートされているサービス](https://github.com/MichaelDeBoey/gatsby-remark-embedder#supported-services)を必ず確認してください。

https://youtu.be/QfcozcbDhNM

## コードのハイライト

コードコンポーネントの多くのオプションをサポートしています。

### タイトル

タイトルを表示するには、コードブロックに追加するだけです。`title=src/myfile.css`

```css title=src/myfile.css
.gatsby::before {
  content: "niceee...";
}
```

### 行番号

そのコードブロックの行番号を追加するには、オプションを渡します `lineNumbers=true`.

```js lineNumbers=true
const rocket = {
  launch: () => console.log("Launching..."),
};

rocket.launch();
```

### React live

冒頭で述べたように、このテーマは`react-live`を使用しているため、React コンポーネントをライブ編集するための遊び場を作成できます。

これを使用するには、コードブロックに`live=true`オプションを追加するだけです。

```jsx lineNumbers=true title=gatsby-config.js live=true
function MyComponent() {
  function handleButtonClick() {
    alert("wowww");
  }

  return <button onClick={handleButtonClick}>Hey, click me!</button>;
}

render(<MyComponent />);
```

> スコープに要素を追加する必要がある場合は、[このガイド](/usage/shadowing#adding-elements-to-react-live-scope)を確認してください。

### ラインハイライト

線を強調表示するには、`highlight`オプションに範囲を渡すことができます。
たとえば`highlight={1-4,6-7,9}`、以下のコードでは、次のようになります。

```jsx highlight={1-4,6-7,9}
function MyComponent() {
  function handleButtonClick() {
    alert("wowww");
  }

  return <button onClick={handleButtonClick}>Hey, click me!</button>;
}

render(<MyComponent />);
```
