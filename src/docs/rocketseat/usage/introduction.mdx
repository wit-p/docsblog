# Introduction

Rocket Docs は、ドキュメント Web サイト用の完全で素晴らしい Gatsby テーマです。
使いやすく、カスタマイズしやすいように開発しました。
他のことを忘れて、重要なことに集中してください

**ドキュメントを書くこと!!**

## 特徴

- ⚛️ Gatsbyv3 の準備ができました。
- 📝 ドキュメント用の MDX;
- 🛣 Yaml ベースのサイドバーナビゲーション。
- 📱 応答性が高く、モバイルフレンドリー。
- 🖥 prism -react-renderer と react-live サポートによるコードの強調表示。
- 🥇 SEO（サイトマップ、schema.org データ、Open Graph、Twitter タグ）。
- 📈 GoogleAnalytics のサポート;
- 📄 カスタムドキュメントスキーマ;
- 🖱 目次;
- ⚡️ オフラインサポートと WebApp マニフェスト。
- その他 🔥
