---
title: Navigation
description: How to use the sidebar.yml file from Rocket Docs.
---

サイドバーにアイテムを追加するには sidebar.yml、config フォルダーの下にファイルを作成する必要があります。例：

```yaml title=sidebar.yml
# Just a simple item
- label: "Home"
  link: "/"
# With a list of items
- label: "With subitems"
  items:
    - label: "My Example"
      link: "/my-example"
```

必要に応じて、外部リンクを追加することもできます。

```yaml
- label: "GitHub"
  link: "https://github.com/jpedroschmitz"
```
